# Jimi Hendrix Tribute Page

A responsive tribute website dedicated to Jimi Hendrix and two of his most influential albums:

- *Are You Experienced?*
- *Axis: Bold as Love*

This project was created as part of a university assignment focused on advanced CSS layout techniques, responsive design, and limited class usage.

---

## Project Overview

The goal of this project was to build a fully responsive multi-section webpage using only HTML and CSS while working under assignment restrictions that limited the number of CSS classes allowed.

Because of these restrictions, the project heavily uses:

- CSS selectors
- `nth-child()`
- Flexbox
- CSS Grid
- Media Queries

The project was later refactored and cleaned up to improve:
- readability
- maintainability
- structure
- responsiveness

---

## Features

- Responsive design for:
  - Desktop
  - Tablet
  - Mobile

- CSS Grid layouts
- Flexbox navigation and content alignment
- Hover image effects
- Custom local fonts using `@font-face`
- Album information sections
- Track lists
- Responsive typography
- Structured layout with reusable CSS variables
